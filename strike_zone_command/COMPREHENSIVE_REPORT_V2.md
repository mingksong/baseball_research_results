## Part 1: ABS 기준 스트라이크 존 제구력 (2025 시즌)

### 1-1. 연구 방법론

#### ABS (Automated Ball-Strike) 존 정의

**출처**: Baseball physics + KBO 규정

```
공 반지름 = 1.45 inches = 0.121 feet
홈플레이트 반폭 = 8.5 inches = 0.708 feet

ABS 스트라이크 존:
- 가로: |plate_x| ≤ 0.829 feet (0.708 + 0.121)
- 세로: (sz_bot - 0.121) ≤ plate_z ≤ (sz_top + 0.121)
```

**이유**: ABS 시스템은 공 중심 위치를 추적하므로, 규정 존 경계에서 공 반지름만큼 벗어나도 공이 존을 스치면 스트라이크 판정

#### 분석 대상

**출처**: `abs_pitcher_command_100plus.csv`

- **2025 시즌 100구 이상 투수**: 236명
- **총 투구수**: 데이터 확인 필요
- **필터링**: `plate_z IS NOT NULL AND sz_bot IS NOT NULL`

## Part 3: Heart/Shadow/Waste 존 세분화 분석 (2025)

### 3-1. 연구 방법론

#### 존 세분화 정의 (Baseball Savant 방식)

**출처**: Baseball Savant Zone 1-9, 11-19 개념

```
Heart Zone (중앙 코어):
- 가로: |plate_x| ≤ 0.4333 ft (0.7083 - 0.2750)
- 세로: (sz_bot + 0.3333) ≤ plate_z ≤ (sz_top - 0.3333)
- 해석: 타자가 가장 잘 치는 중앙 영역

Shadow Zone (엣지 링):
- 규정존 안쪽 엣지 + 규정존 바깥 ±pad
- 가로 pad: 3.3 inches (0.2750 ft)
- 세로 pad: 4.0 inches (0.3333 ft)
- 해석: 스트라이크 판정 경계 영역

Waste Zone:
- Shadow 밖 모든 구역
- 해석: 명백한 볼, 헛스윙 유도 목적

All Strike = Heart + Shadow
```

## Part 4: CSW% 및 투구 결과 분석 (2025)

### 4-1. 연구 방법론

#### pitch_result 매핑

**출처**: 데이터베이스 직접 확인

| 코드 | 의미 | 스트라이크 여부 |
|------|------|----------------|
| S | Strike (룩킹) | ✅ |
| W | Whiff (헛스윙) | ✅ |
| F | Foul (파울) | ✅ |
| H | Hit (인플레이) | ❌ |
| B | Ball (볼) | ❌ |

#### CSW% 정의

```
CSW% = (룩킹 스트라이크 + 헛스윙) / 전체 투구 × 100
     = (S + W) / Total × 100
```

### 4-2. 리그 전체 CSW% 및 투구 결과

|------|------|
| **CSW%** | **26.92%** |
| Looking Strike% (S) | **17.63%** |
| Whiff% (W) | **9.29%** |
| Called Strike% (S+W+F) | **43.94%** |
| ABS Zone% | **46.25%** |

**데이터 기반 사실**:
1. 룩킹 + 헛스윙 = 26.92% (10구 중 2.7구)
2. 파울 포함 시 43.94%
3. Zone% 46.25% vs Called% 43.94% = -2.31%p 차이
