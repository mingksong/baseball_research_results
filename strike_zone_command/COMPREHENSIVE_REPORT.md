## 🎯 Part 1: ABS 기준 제구력 분석 (2025 시즌)

### 1-1. 연구 배경

**ABS 스트라이크 존 정의**:
```
규정 스트라이크 존 + 야구공 반지름(1.45인치)

- 세로: sz_bot - 0.121 ≤ plate_z ≤ sz_top + 0.121
- 가로: |plate_x| ≤ 0.829 feet (홈플레이트 반폭 + 공 반지름)
```

**이유**: ABS 센서는 공 중심 위치를 추적하므로, 규정 존에서 공 반지름만큼 벗어난 포인트도 공이 존을 스치면 스트라이크

## 🎯 Part 4: Heart/Shadow/Waste 존 정밀 분석 (2025 시즌)

### 4-1. 존 재정의: Baseball Savant 방식

**기존 ABS 존의 한계**:
- 규정존 + 공 반지름(0.121ft) = 단일 기준
- 중앙 vs 엣지 구분 불가

**새로운 3단계 존**:
```
Heart Zone (중앙 코어):
  - 가로: |plate_x| ≤ 0.4333 ft (규정존 - 3.3 inches)
  - 세로: sz_bot + 0.3333 ≤ plate_z ≤ sz_top - 0.3333

Shadow Zone (엣지 링):
  - 규정존 안쪽 엣지 + 규정존 바깥 ±pad
  - 가로: |plate_x| ≤ 0.9833 ft (규정존 + 3.3 inches)
  - 세로: sz_bot - 0.3333 ≤ plate_z ≤ sz_top + 0.3333

Waste Zone (명백한 볼):
  - Shadow 밖 모든 구역
```

**출처**: `heart_shadow_by_count.csv`, `heart_shadow_by_runners.csv`, `heart_shadow_by_pitch_type.csv`

| **Shadow** (엣지 링) | **41.56%** | 대부분이 엣지 공략 |
| **Waste** (볼) | **41.77%** | 거의 절반이 볼 |

- 스트라이크의 70%가 Shadow(엣지) - 안타 방지 전략
- Waste(볼) 42% - 헛스윙 유도 적극 활용

|------|---------|------|
| 일관된 개선 (3년 연속 상승) | **4명** | 3.1% |
| 일관된 하락 (3년 연속 하락) | **49명** | 38.6% |

**개선 4명**:
1. 김유성: 40.2% → 44.7% → 49.3% (+9.0%p)

4. 윤석원: 50.1% → 50.3% → 51.0% (+0.9%p)

**최악의 하락**:
- 정우영: 52.6% → 43.6% → 27.0% (**-25.6%p**)
- 김대현: 59.5% → 42.3% → 37.9% (-21.6%p)

**인사이트**:
- **제구력은 개선보다 하락이 12배 더 쉽다**
- 현 수준 유지도 어렵다
- 3년 연속 개선은 극소수(3.1%)
- 한번 하락 시작하면 급격히 악화 (정우영 케이스)

## 4. 상황별 제구력: "압박받으면 더 떨어진다"

### 4-1. 카운트별 변화

**출처**: `heart_shadow_by_count.csv`, `called_strike_by_count.csv`

| 카운트 | Heart% | Shadow% | All Strike% | CSW% |
|--------|--------|---------|-------------|------|
| **First Pitch** | 18.88% | 44.16% | 63.04% | 38.65% |
| **Two Strikes** | 13.82% | 37.94% | 51.77% | 16.90% |
| **Three Balls** | 21.27% | 47.04% | 68.31% | 37.82% |

**변화**:
- 초구 → 투스트라이크: Heart **-5.06%p**, Shadow **-6.22%p**
- 투스트라이크에서 CSW% 16.90% (최저)
- 3볼에서 All Strike 68.31% (최고) - 볼넷 회피

**전략 변화**:
1. **투스트라이크**: 존 회피 → Waste로 헛스윙 유도
2. **3볼**: 적극적 존 공략 → 볼넷 방지
3. **Ahead (유리)**: Heart 14.88% (최저) → 중앙 피하고 엣지 활용

### 4-2. 주자 상황별 변화

**출처**: `heart_shadow_by_runners.csv`

| 주자 | Heart% | Shadow% | All Strike% |
|------|--------|---------|-------------|
| **Empty** | 17.67% | 43.25% | 60.92% |
| **RISP** | 15.67% | 40.71% | 56.38% |

**변화**:
- Empty → RISP: Heart **-2.00%p**, Shadow **-2.54%p**
- 득점권에서 중앙 공략 회피 (장타 방지)
- 클러치 상황 = 제구보다 안전 우선

### 4-3. 구종별 제구율

**출처**: `heart_shadow_by_pitch_type.csv`

| 구종 | Heart% | Shadow% | All Strike% |
|------|--------|---------|-------------|
| **Fastball** | 19.51% | 45.47% | 64.98% |
| **Breaking** | 15.28% | 39.16% | 54.45% |
| **OffSpeed** | 12.07% | 37.81% | 49.87% |

**차이**: Fastball vs OffSpeed = **15.11%p**

**인사이트**:
- 직구가 가장 제구하기 쉬움 (64.98%)
- 체인지업/포크는 중앙 회피 (12.07%)
- OffSpeed 중앙 공략 = 높은 위험

## 7. 종합 인사이트 및 결론

### 7-1. KBO 제구력의 3대 특징

#### ① 절반도 안 된다 (46.25%)
- 평균 투수: 10구 중 4.6구만 존
- 정중앙(Heart): 10구 중 1.7구
- CSW%: 10구 중 2.7구

#### ② 개선은 거의 불가능 (1:12)
- 3년 연속 개선: 4명 (3%)
- 3년 연속 하락: 49명 (38%)
- 한번 하락 시작하면 급격히 악화

#### ③ 중앙 회피 전략 (Heart 16.66%)
- 스트라이크의 70%가 엣지(Shadow)
- 압박 상황에서 더 회피
- 득점권: Heart -2.00%p

### 7-3. 실전 적용 인사이트

#### 팬들에게
1. **제구력 = 생존의 필수 조건**
   - 100구 만에 2군 가는 투수들 (45.16%)
   - 1000구 살아남은 투수들 (47.36%)
   - 차이는 겨우 2.20%p

2. **"네모 안" 기대는 비현실적**
   - 평균 46.25%
   - 엘리트도 65% 수준
   - 정중앙은 17%만

3. **제구력은 거의 개선 안 됨**
   - 현 수준 유지가 목표
   - 하락 시작하면 급격히 악화

#### 구단 운영진에게
1. **제구력 기준 설정**
   - 신인 최소 기준: All Strike 50%+
   - 1군 유지 기준: 55%+
   - 엘리트 기준: 65%+

2. **조기 경보 시스템**
   - 시즌 중 -5%p 하락 시 집중 관리
   - 연속 하락 시 근본 문제 점검
   - 정우영 케이스 (-25.2%p) 방지

3. **2024년 급락 원인 규명**
   - 리그 전체 -3%p
   - 공인구/ABS 영향 분석
   - 타자 선구안 변화 연구

#### 코칭스태프에게
1. **현실적 목표 설정**
   - 제구 "개선" 기대 X
   - 현 수준 "유지"가 성공
   - Heart보다 Shadow 활용 훈련

2. **상황별 전략**
   - 투스트라이크: Waste 활용 (16.90% CSW)
   - 3볼: 적극 존 공략 (68.31%)
   - 득점권: 중앙 회피 (Heart -2%p)

3. **구종별 차별화**
   - Fastball: 제구 훈련 집중
   - OffSpeed: 중앙 회피 교육
   - 직구 제구 없으면 변화구도 안 됨
