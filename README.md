## 📊 연구 주제

### 1. [원태인 투수 상세 분석](won_taein_analysis/)
**기간**: 2021-2025 시즌 (5시즌)
**목적**: 원태인 투수의 로케이션, 피로도, 상대별 성적 종합 분석

**주요 분석 내용**:
- 로케이션 분석 (구종별, 타자 좌우별)
- 피로도 분석 (등판 간격, 워크로드, 구속 변화)
- 상대별 분석 (팀별, 타자별 성적)
- 물리량 기반 홈플레이트 위치 계산

**결과물**:
- 최종 보고서: [FINAL_REPORT.md](won_taein_analysis/report/FINAL_REPORT.md)
- 27개 시각화 차트
- 25개 CSV 데이터 파일

**핵심 발견사항**:
- 5시즌 134경기 등판, 12,736개 투구
- 주력 구종: 직구(44.2%), 체인지업(24.4%), 슬라이더(23.0%)
- 좌타자 상대 체인지업 29.83%, 우타자 상대 슬라이더 26.62% 집중 사용
- 2024-2025 시즌 plate_z 디폴트 값 문제 발견 및 물리량 기반 보정

### 2. [2025 투수 레파토리 위치 분석](pitcher_repertoire_2025/)
**대상**: 2025년 600구 이상 투수 126명
**목적**: 각 투수의 주요 레파토리 평균 위치 시각화

**주요 분석 내용**:
- 100구 이상 던진 모든 구종 분석
- 좌타자/우타자 구분
- 구종별 평균 로케이션 표시

**결과물**:
- 126개 투수별 차트 (PNG)
- 파일명 형식: `{팀코드}_{pcode}_{선수이름}_avg_location.png`
- 투수 목록 CSV

**핵심 발견사항**:
- 600구 이상 투수 126명 분석 완료
- 투수별 2~4개 주요 레파토리 확인
- 타자 좌우별 뚜렷한 로케이션 전략 차이 발견

### 3. [디아즈 홈런 타구 로케이션 분석 (2025)](diaz_homerun_2025/)
**대상**: 디아즈 (삼성 라이온즈)
**목적**: 2025년 시즌 홈런을 친 투구의 로케이션 분석

**주요 분석 내용**:
- 홈런 50개 (좌투수 15개, 우투수 35개)
- 투수 좌우 구분 스캐터플롯
- 구종별 홈런 발생 위치

**결과물**:
- 좌투/우투 구분 로케이션 차트
- 홈런 투구 데이터 CSV
- 구종별 홈런 통계

**핵심 발견사항**:
- 2025년 총 50개 홈런 (좌투수 15개, 우투수 35개)
- 홈런의 50%가 직구에서 발생
- 포크볼 홈런 16%, 슬라이더 홈런 14%
- 우투수 상대 홈런이 2.3배 더 많음

### 4. [스트라이크 존 제구력 연구 (2021-2025)](strike_zone_command/)
**기간**: 2021-2025 시즌 (5시즌)
**대상**: 236명 투수 (100구 이상)
**목적**: KBO 투수들의 스트라이크 존 제구력 종합 분석

**주요 분석 내용**:
- ABS 존 기준 제구율 분석 (규정 존 + 공 반지름)
- 시즌별 추이 분석 (2021-2025)
- Heart/Shadow/Waste 존 분포
- CSW% (Called Strike + Whiff %) 분석
- 상황별 제구력 변화 (카운트, 주자, 구종)

**결과물**:
- 최종 보고서: [COMPREHENSIVE_REPORT_V2.md](strike_zone_command/COMPREHENSIVE_REPORT_V2.md)
- 20개 CSV 데이터 파일
- 5개 JSON 분석 결과

**핵심 발견사항**:
- **평균 제구율 47.06%** - 투구의 절반도 존에 안 들어감
- **2024년 미스터리 급락** - 전년 대비 -3.00%p 하락
- **제구력 개선 극히 드물다** - 3년 연속 개선 vs 하락 = 1:12 (4명 vs 49명)
- **생존편향 존재** - 100-200구 투수 45.16% vs 1000구+ 투수 47.36% (+2.20%p)
- **상황별 전략 변화** - 투스트라이크 시 Heart -5.06%p, Shadow +5.06%p
- **구종별 차이** - Fastball 52.77% vs OffSpeed 38.52% (차이 14.25%p)

**주요 인사이트**:
- "네모 안으로 던지기"는 야구에서 가장 어려운 기술
- KBO 투수들은 정중앙(Heart) 16.66%만 사용 → 의도적 중앙 회피
- 순수 스트라이크(CSW)는 26.92% → 10구 중 2.7개만
- 제구력은 타고난 능력이며 개선이 거의 불가능
