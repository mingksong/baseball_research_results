# 디아즈 홈런 타구 로케이션 분석 (2025)

## 연구 개요
2025년 시즌 디아즈(삼성 라이온즈) 선수의 홈런 타구에 대한 투구 로케이션 분석

## 연구 대상
- **선수**: 디아즈 (pcode: 54400)
- **팀**: 삼성 라이온즈 (SS)
- **시즌**: 2025년 정규시즌
- **총 홈런**: 50개
  - 좌투수 상대: 15개
  - 우투수 상대: 35개

## 데이터 추출 방법
- `plate_appearances` 테이블에서 `result_type = 'HR'`인 타석 식별
- 해당 타석의 마지막 투구(홈런을 친 투구) 위치 데이터 추출
- 투수의 좌우 구분 (phand)

## 주요 발견사항

### 구종별 홈런 분포
1. **직구**: 25개 (50%)
2. **포크**: 8개 (16%)
3. **슬라이더**: 7개 (14%)
4. **체인지업**: 5개 (10%)
5. **투심**: 2개 (4%)
6. **커브**: 2개 (4%)
7. **커터**: 1개 (2%)

### 투수 좌우별 특성
- 우투수 상대 홈런이 2.3배 더 많음 (35개 vs 15개)
- 주로 직구에서 홈런 발생 (전체의 50%)
- 포크볼도 상당수 홈런으로 연결 (16%)
