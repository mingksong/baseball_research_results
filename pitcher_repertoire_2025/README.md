# 2025 시즌 투수 레파토리 위치 분석

## 연구 목적
2025년 시즌 600구 이상 투구한 투수들의 주요 구종 평균 로케이션을 타자 좌우별로 분석

## 분석 대상
- **시즌**: 2025 정규시즌
- **기준**: 600구 이상 투구
- **대상 투수**: 126명
- **구종**: 투수별 100구 이상 투구한 모든 구종

## 출력 형식
- 파일명: `{teamcode}_{pcode}_{선수이름}_avg_location.png`
- 내용: 좌타자/우타자 상대 주요 구종별 평균 위치 scatter plot

- 100구 이상 투구한 모든 구종 포함
- 스트라이크존 오버레이
